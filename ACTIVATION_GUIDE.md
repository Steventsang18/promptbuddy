# PromptBuddy 激活指南

## ⚠️ 当前限制

OpenClaw 的技能机制需要 AI 判断是否触发。新会话或新 Agent 可能需要手动确认技能状态。

---

## 🚀 快速激活方式

### 方式 1：新建对话（推荐）

在当前对话中，**新建对话**即可自动加载最新技能。

### 方式 2：询问激活

在对话中发送：
```
你安装了 promptbuddy 技能吗？请激活使用
```

AI 会检查并激活 PromptBuddy。

### 方式 3：直接触发

直接发送问题，如果技能已正确安装，会自动触发：
```
帮我写一个产品介绍
```

---

## 📋 多 Agent 场景

如果使用多个 Agent（如 sunny, xiaohong 等），每个 Agent 需要：

1. **新建对话**，或
2. **询问激活**

### 一键安装脚本

运行以下命令，确保所有 Agent 都安装 PromptBuddy：

```bash
bash ~/.openclaw/workspace/skills/promptbuddy/install-all-agents.sh
```

---

## ✅ 验证技能是否生效
