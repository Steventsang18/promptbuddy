# PromptBuddy 项目记忆

> 本文档用于记录项目开发进度，方便后续无缝对接继续完善。

## 项目概述

**项目名称**: PromptBuddy - 智能 Prompt 伙伴
**Slogan**: 要AI真懂你 | OnlyPromptBuddy

**目标**: 打造一个无需触发词、自动识别用户需求、智能选择最佳模板、支持用户偏好学习的 Prompt 优化系统。

**核心理念**: 
- 用户正常说话，系统自动处理
- 越用越懂你（渐进式学习）
- 混合反馈机制（显式+隐式）

## 技术架构

```
用户输入
    │
    ▼
┌─────────────────────────────────────────────────────┐
│                  需求识别层                          │
│  NeedDetector                                       │
│  - 任务词检测（帮我、请、写、生成...）              │
│  - 问句模式检测（如何、怎么、为什么...）            │
│  - 排除模式过滤（天气、问候...）                    │
└─────────────────────────────────────────────────────┘
    │ need_optimization=True
    ▼
┌─────────────────────────────────────────────────────┐
│                  意图识别层                          │
│  HybridIntentDetector                               │
│  ┌─────────────┐    ┌─────────────┐                │
│  │ 关键词引擎  │    │ 语义引擎    │                │
│  │ (NLP)       │    │ (TF-IDF)    │                │
│  └──────┬──────┘    └──────┬──────┘                │
│         │    融合决策    │                         │
│         └───────┬────────┘                          │
│                 ▼                                   │
│         智能融合策略                                 │
└─────────────────────────────────────────────────────┘
    │ intent + confidence
    ▼
┌─────────────────────────────────────────────────────┐
│                  决策层                              │
│  AutoDecisionEngine                                 │
│  - 结合用户偏好                                      │
│  - 自适应阈值调整                                    │
│  - 历史反馈影响                                      │
└─────────────────────────────────────────────────────┘
    │ template
    ▼
┌─────────────────────────────────────────────────────┐
│                  模板生成层                          │
│  Template Engine                                    │
│  - base: 基础模板                                   │
│  - cot: 思维链模板                                  │
│  - decompose: 任务分解模板                          │
│  - criticize: 自我批评模板                          │
└─────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────┐
│                  反馈系统                            │
│  FeedbackCollector                                  │
│  - 显式反馈（用户选择）                              │
│  - 隐式反馈（行为观察）                              │
│  - 自适应调整                                        │
└─────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────┐
│                  用户偏好存储                        │
│  UserProfileManager                                 │
│  - 使用统计                                          │
│  - 模板偏好                                          │
│  - 动态阈值                                          │
└─────────────────────────────────────────────────────┘
```
