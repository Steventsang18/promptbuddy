# PromptBuddy

**Make AI Truly Understand You**

PromptBuddy 是您的智能 Prompt 伙伴。当用户提出问题时，系统会：

1. **自动识别**需求类型
2. **智能选择**最佳模板
3. **透明展示**工作过程
4. **生成优化**的回复

## 工作流程

当用户提出需求时：

```
┌─────────────────────────────────────┐
│ 🤖 PromptBuddy 已优化               │
│ 识别: [问题类型]                    │
│ 模板: [模板名称]                    │
│ 置信度: [0.00-1.00]                 │
└─────────────────────────────────────┘

[开始回答...]
```

## 四种模板

| 模板 | 类型 | 说明 |
|-----|------|------|
| CoT | 推理型 | 逐步深度思考 |
| Decompose | 分解型 | 任务步骤分解 |
| Criticize | 检查型 | 质量审核改进 |
| Base | 通用型 | 简洁专业回答 |

## 使用方式

用户正常说话即可，系统自动处理：

```
用户: "火箭如何上天？"

系统输出:
┌─────────────────────────────────────┐
│ 🤖 PromptBuddy 已优化               │
│ 识别: 推理型问题                    │
│ 模板: CoT（思维链）                 │
│ 置信度: 0.43                        │
└─────────────────────────────────────┘

[基于思维链模板的回答...]
```

## 特殊情况

- **简单查询**（天气、问候）：跳过优化，直接回答
- **低置信度**：可能需要用户确认模板选择

## 渐进学习

- 新用户：更谨慎，低置信度会询问
- 学习期：根据用户偏好调整
- 成熟期：几乎全自动，完全个性化
